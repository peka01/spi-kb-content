# Digital inlämning av INK3 och INK4 via API

Nu blir det ännu enklare att sköta deklarationen för ditt handelsbolag eller din förening. Från och med deklarationsperioden 2025-P1 kan du skicka in inkomstdeklaration 3 och 4 direkt till Skatteverket via API – precis lika smidigt som för aktiebolag.

Genom att använda den digitala kopplingen (API) slipper du hantera filer manuellt, vilket minskar risken för fel och ger dig snabbare bekräftelse på att din deklaration har tagits emot.

## Vad innebär nyheten?
Tidigare har digital inlämning via API främst varit tillgänglig för aktiebolag (INK2). Nu har vi utökat denna möjlighet i Spiris så att även du som företräder följande företagsformer kan arbeta mer effektivt:
*   **Handelsbolag (HB)** – Inkomstdeklaration 3
*   **Stiftelser och ideella föreningar (ST/IF)** – Inkomstdeklaration 4

Du kan nu välja mellan att skicka informationen direkt via den digitala kopplingen eller att ladda ner traditionella **SRU-filer**.

## Förutsättningar
För att kunna använda den nya funktionen behöver du:
*   Arbeta i deklarationsperioden **2025-P1** eller senare.
*   Ha ett aktivt abonnemang för deklarationsfunktionerna i Spiris.

## Så här gör du för att lämna in digitalt

Följ dessa steg för att skicka in din deklaration direkt till Skatteverket:

1.  Gå till vyn för inlämning av deklarationen i din ekonomiplattform.
2.  Kontrollera att alla uppgifter i din **Inkomstdeklaration 3** eller **Inkomstdeklaration 4** är korrekta.
3.  Under avsnittet för inlämningsmetod väljer du alternativet **Digital inlämning via API**.
4.  Klicka på knappen **Skicka till Skatteverket**.
5.  Du slutför sedan inlämningen genom att signera deklarationen på Skatteverkets webbplats (Mina sidor) när överföringen är klar.

## Resultat
När du har skickat in deklarationen via API får du direkt en statusuppdatering i Spiris som bekräftar att överföringen lyckats. Statusen för din deklaration ändras till **Inskickad via API**.

## Fördelar med API-inlämning
*   **Säkrare:** Ingen risk att fel filer laddas upp av misstag.
*   **Snabbare:** Informationen landar direkt hos Skatteverket utan mellansteg.
*   **Enklare:** Du får en tydlig kvittens direkt i plattformen.

## Vanliga frågor (Felsökning)
**Varför ser jag inte alternativet för API-inlämning?**
Kontrollera att du befinner dig i rätt deklarationsperiod (2025-P1). Om du arbetar med ett tidigare beskattningsår är det fortfarande inlämning via **SRU-filer** som gäller.

**Måste jag fortfarande logga in hos Skatteverket?**
Ja, själva signeringen av deklarationen sker av säkerhetsskäl på Skatteverkets tjänst **Mina sidor**. Spiris ser till att all data förs över dit så att den ligger klar för dig att skriva under.
