# Export and archive data for multiple years in {{resource:common_product_name_variable_taxation_year_end_closing}}

In this article, we describe how to use the new function to export and archive the company's documentation for one or more years simultaneously. This is a convenient way to ensure that you have all necessary documentation saved digitally for the future.

## Introduction
To fulfill legal archiving requirements or to have a local backup before a subscription is terminated, you can now export all relevant data from **{{resource:common_product_name_variable_taxation_year_end_closing}}**. The function creates an organized ZIP file with PDF documents for the years and periods you select.

## Prerequisites
* You have an active subscription for {{resource:common_product_name_variable_taxation_year_end_closing}}.
* The function is also available for those who have the company in read-only mode.

## How to export data

1. Go to the menu and select ****{{resource:Menu_title_Maintenance}}**** – ****{{resource:ExportData_title_ExportData}}****.
2. Under the export heading, select the **Documentation** option. 
3. In the list ****{{resource:Startpage_hdr_FinancialYear}}****, select the years you want to include in the export. You can select a single year, several specific years, or select **All years**.
4. Click the button ****{{resource:Maintenance_ExportData_hdr_Export}}****.
5. A progress bar is displayed while the export is being prepared. You can continue working in the platform in the meantime.
6. When the export is complete, you will receive a notification. Click the link in the notification to download the ZIP file.

## Result
When the export is complete, a ZIP file is downloaded to your computer. The file is structured as follows:
* **Main folders**: Named after the respective financial year.
* **Subfolders**: Each year contains folders for the periods that have had activity.
* **Documents**: In each period folder, you will find PDF files for all relevant documents, such as reconciliations, tax calculations, and annual report documents.

**Note**: To keep the export relevant, only periods with automatic checks, marked accounts, or comments are included.

## Good to know
* **Tooltips**: If you hover your mouse over the information symbol next to the different document types, you can see exactly what is included in the export.
* **Status**: The export button is disabled while a run is in progress to prevent duplicate exports.
* **Error handling**: If a document cannot be generated (for example, due to an unusual file format), the export will still complete. You will then receive a message that the export was partially successful, and in the ZIP file, you will find information about which document is missing and why.

By using this function, you save time and can feel confident that your digital archiving is handled in a structured and secure way in {{resource:Common_Company_Name_Short}}.
