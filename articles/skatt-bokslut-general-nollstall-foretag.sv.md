# Nollställ företag i Skatt & Bokslut

## Översikt
Ibland blir det fel, och det är helt okej. Kanske har du importerat fel fil eller bara vill börja om på ny kula med ett räkenskapsår. Med den här funktionen kan du enkelt radera all data för det aktiva företaget och året, så att du kan börja om från början.

Tänk på att den här handlingen är permanent och **inte går att ångra**. All information för det valda året, som grunduppgifter och importerad data, kommer att raderas.

## Så här nollställer du ett företag
Följ de här stegen för att radera all data för det aktiva företaget.

1.  Gå till menyn **Arkiv** högst upp på sidan.
2.  Välj alternativet **Nollställ företag**.
3.  En varningsruta visas för att säkerställa att du inte raderar allt av misstag. För att bekräfta, skriv in företagets namn exakt som det står i rutan.
4.  Klicka på knappen **Nollställ**.

Nu är all data för företaget raderad och du kan börja om, till exempel genom att importera en ny SIE-fil.
