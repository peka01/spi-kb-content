# Digital inlämning av INK3 och INK4 via API

Här hittar du svar på de vanligaste frågorna om den nya möjligheten att lämna in inkomstdeklaration 3 och 4 digitalt direkt till Skatteverket.

### Vad innebär den nya funktionen för digital inlämning?
Nu blir det ännu enklare att sköta deklarationen för fler verksamhetsformer i vår plattform. Tidigare har du kunnat skicka in inkomstdeklaration 2 (för aktiebolag) via API, och nu utökar vi denna smidiga lösning till att även gälla INK3 och INK4. Det innebär att du skickar informationen direkt till Skatteverket med ett klick, istället för att hantera filer manuellt.

### Vilka verksamheter kan använda den nya API-inlämningen?
Den nya funktionen är tillgänglig för:
*   **Handelsbolag (HB)** som lämnar in INK3.
*   **Stiftelser och ideella föreningar (ST/IF)** som lämnar in INK4.

### Från och med när kan jag börja använda API-inlämning för dessa blanketter?
Du kan använda den digitala inlämningen via API från och med taxeringsperioden **2025-P1**. 

### Hur väljer jag att skicka in via API istället för SRU?
När det är dags att lämna in deklarationen går du till sidan för inlämning. Där har du nu möjlighet att välja mellan två metoder:
1.  **Inkomstdeklaration via API**: Detta är det snabbaste sättet där uppgifterna förs över direkt till Skatteverket.
2.  **SRU**: Den traditionella metoden där filer skapas för filöverföring.

Vi rekommenderar att använda API-alternativet för en mer sammanhållen och säker upplevelse.

### Varför är det bättre att använda API än SRU?
Att använda API innebär en mer direkt kommunikation med Skatteverket. Du får snabbare bekräftelse på att informationen har tagits emot och du slipper ladda ner och ladda upp filer manuellt. Det minskar risken för fel och sparar värdefull tid i ditt arbete.

### Behöver jag ändra något i mitt abonnemang för att få tillgång till detta?
Nej, funktionen ingår som en del av de löpande uppdateringarna i din ekonomiplattform från Spiris. Så länge du har ett aktivt abonnemang som täcker deklarationsarbete för dessa bolagsformer kommer alternativen finnas tillgängliga på inlämningssidan.
