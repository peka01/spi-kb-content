# Archive and export documentation for fiscal years

In Spiris, you can export and archive the documentation for your fiscal years. This is a secure way to ensure that all supporting documentation is saved and accessible even after a year has been closed, which simplifies your oversight and long-term storage.

## How to do it

1. Go to the **Fiscal Years** view under the accounting settings.
2. Locate the fiscal year you want to archive.
3. Select the function to export documentation.
4. The system will now begin compiling your documents. You can continue working in the platform in the meantime.
5. Once the export is complete, you will receive a notification confirming that the file is ready for download.

## Information about export status

We have updated how the system informs you about the export results to give you better control:

*   **Export complete:** All documentation has been compiled correctly into a ZIP file.
*   **Export partially completed:** If the system for any reason could not include every single document, you will now receive a specific notification that the export is partially completed. This ensures you are immediately alerted if any individual supporting document needs to be checked manually, rather than assuming everything went flawlessly.

## Tip
> Remember that the export function is not available for fiscal years created via fully manual entry or via SIE1 files. This is because these import methods do not contain the detailed documentation required for a complete export.
