# Now easier to file tax returns for trading partnerships and foundations!

**Starting from the 2025-P1 tax filing period, you can submit income tax returns 3 and 4 directly to the Swedish Tax Agency via API.**

Previously, direct digital submission was primarily available for limited companies (INK2). The function is now being expanded to include trading partnerships (HB), foundations (ST), and non-profit organizations (IF). This means you avoid manual file handling and can submit the data in the same smooth workflow as for limited companies.

### What does this news mean for you?

Instead of manually downloading and uploading SRU files, the information is now sent encrypted and securely directly from {{resource:Common_Company_Name_Short}} to the Swedish Tax Agency.

**The main benefits:**
*   **Fast and easy:** Submit the tax return with just a few clicks.
*   **Reduced margin of error:** The API connection minimizes risks associated with manual file handling.
*   **Full control:** You can see directly in {{resource:common_product_name_variable_taxation_year_end_closing}} that the submission has been received.

### How it works

When filing for the 2025-P1 period, you can choose the new workflow on the submission page:

1.  Go to the tax return submission view.
2.  Select **Digital submission via API** (alternatively traditional **SRU files**).
3.  The data for **Trading Partnership (INK3)** or **Foundation/Non-profit organization (INK4)** is sent directly to the Swedish Tax Agency.
4.  You receive a confirmation and status update directly in the view.

### Who can use the function?

The update is available to all users who manage trading partnerships, foundations, or non-profit organizations. The function is automatically activated for financial years filed from P1 2025 onwards.

With a fully digital workflow, you can spend less time on tax return files and more time on your business!
