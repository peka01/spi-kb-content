# Vanliga frågor om koppling mellan Skatt & Bokslut och Fortnox

Här har vi samlat svar på de vanligaste frågorna om hur du använder kopplingen mellan Fortnox och Skatt & Bokslut för att göra ditt bokslutsarbete smidigare.

### Kan jag hämta data från Fortnox direkt till Skatt & Bokslut?
Ja, genom den nya kopplingen kan du hämta bokföringsdata direkt från Fortnox. Istället för att manuellt ladda ner och ladda upp filer kan du nu föra över SIE4-data med bara några klick. Detta gör att du snabbt kommer igång med ditt bokslut eller din årsredovisning i **Skatt & Bokslut**.

### Vilken information kan jag överföra?
Funktionen låter dig hantera flera delar av bokslutet på ett effektivt sätt:
*   **Bokföringsdata**: Hämta aktuella siffror via SIE4-format.
*   **Reskontra**: Hantera underlag för kund- och leverantörsfakturor.
*   **Verifikationer**: Du kan skicka färdiga verifikationer från **Skatt & Bokslut** direkt tillbaka till Fortnox.

### Hur aktiverar jag kopplingen?
Du aktiverar funktionen genom att gå till inställningarna för import i din ekonomiplattform. Välj alternativet för att hämta data från Fortnox och följ instruktionerna på skärmen för att logga in och godkänna kopplingen. När det är klart kan du börja hämta information direkt till ditt pågående arbete.

### Hur sparar jag tid med denna funktion?
Genom att koppla ihop dina verktyg slipper du dubbelarbete och manuell hantering av filer. Eftersom du kan skicka tillbaka verifikationer direkt till Fortnox säkerställer du att din löpande bokföring stämmer överens med ditt bokslut utan att behöva knappa in informationen på nytt. Det ger dig mer tid över till rådgivning och analys istället för administration.

### Vem kan använda kopplingen?
Alla användare som har ett aktivt abonnemang för **Skatt & Bokslut** och använder Fortnox kan använda den här funktionen. Det krävs inga extra avtal för att komma igång, utan funktionen finns tillgänglig direkt i plattformen.
