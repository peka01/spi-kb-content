# Nu enklare att deklarera för handelsbolag och stiftelser!

**Från deklarationsperioden 2025-P1 kan du lämna in inkomstdeklaration 3 och 4 direkt till Skatteverket via API.**

Tidigare var direkt digital inlämning främst tillgänglig för aktiebolag (INK2). Nu utökas funktionen till att omfatta handelsbolag (HB), stiftelser (ST) och ideella föreningar (IF). Det innebär att du slipper manuell filhantering och kan skicka in uppgifterna i samma smidiga flöde som för aktiebolag.

### Vad innebär nyheten för dig?

Istället för att ladda ner och ladda upp SRU-filer manuellt, skickas informationen nu krypterat och säkert direkt från {{resource:Common_Company_Name_Short}} till Skatteverket.

**De främsta fördelarna:**
*   **Snabbt och enkelt:** Skicka in deklarationen med bara några klick.
*   **Minskad felmarginal:** API-kopplingen minimerar risker vid manuell filhantering.
*   **Full kontroll:** Du ser direkt i {{resource:common_product_name_variable_taxation_year_end_closing}} att inlämningen tagits emot.

### Så här fungerar det

Vid deklaration för perioden 2025-P1 kan du välja det nya flödet på inlämningssidan:

1.  Gå till vyn för inlämning av deklaration.
2.  Välj **Digital inlämning via API** (alternativt traditionella **SRU-filer**).
3.  Uppgifterna för **Handelsbolag (INK3)** eller **Stiftelse/Ideell förening (INK4)** skickas direkt till Skatteverket.
4.  Du får en bekräftelse och statusuppdatering direkt i vyn.

### Vem kan använda funktionen?

Uppdateringen är tillgänglig för alla användare som hanterar handelsbolag, stiftelser eller ideella föreningar. Funktionen aktiveras automatiskt för räkenskapsår som deklareras från och med P1 2025.

Med ett helt digitalt flöde kan du lägga mindre tid på deklarationsfiler och mer tid på din verksamhet!
