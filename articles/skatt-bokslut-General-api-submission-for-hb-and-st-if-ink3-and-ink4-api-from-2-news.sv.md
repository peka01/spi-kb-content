# Nu blir det ännu enklare att deklarera för handelsbolag och stiftelser!

**Vi på Spiris arbetar ständigt för att förenkla din vardag och göra de administrativa måstena så smidiga som möjligt. Nu tar vi nästa steg i att digitalisera deklarationsprocessen – från och med deklarationsperioden 2025-P1 kan du lämna in inkomstdeklaration 3 och 4 direkt till Skatteverket via API.**

Tidigare har möjligheten till direkt digital inlämning främst varit tillgänglig för aktiebolag (INK2). Nu utökar vi denna funktion till att även omfatta handelsbolag (HB), stiftelser (ST) och ideella föreningar (IF). Det innebär att du slipper krånglig filhantering och kan skicka in dina uppgifter med samma enkla flöde som du är van vid för aktiebolag.

### Vad innebär nyheten för dig?

För dig som driver eller sköter bokföringen åt ett handelsbolag eller en förening betyder detta mindre manuellt arbete och ökad trygghet. Istället för att ladda ner SRU-filer och manuellt ladda upp dem på Skatteverkets webbplats, skickas informationen nu krypterat och säkert direkt från Spiris plattform.

**De främsta fördelarna:**
*   **Snabbt och enkelt:** Skicka in deklarationen med bara några få klick.
*   **Minskad felmarginal:** Genom att använda den direkta kopplingen (API) minskar risken för fel som kan uppstå vid manuell filhantering.
*   **Full kontroll:** Du ser direkt i plattformen att inlämningen har tagits emot av Skatteverket.

### Så här fungerar det

När det är dags att deklarera för perioden 2025-P1 kommer du att märka att du får fler valmöjligheter på inlämningssidan.

1.  Gå till vyn för inlämning av deklaration.
2.  Du kan nu välja mellan att använda den nya, smidiga **Digital inlämning via API** eller den traditionella metoden med **SRU-filer**.
3.  Väljer du API-metoden skickas dina uppgifter för **Handelsbolag (INK3)** eller **Stiftelse/Ideell förening (INK4)** direkt till Skatteverkets mottagning.
4.  När inlämningen är klar får du en bekräftelse direkt i vyn, och statusen uppdateras till att vara inskickad.

### Vem kan använda funktionen?

Denna uppdatering är tillgänglig för alla användare i Spiris som hanterar företagstyperna handelsbolag, stiftelser eller ideella föreningar. Funktionen aktiveras automatiskt för räkenskapsår som deklareras från och med den första perioden 2025.

Vi är glada över att kunna erbjuda ett ännu mer sammanhållet och digitalt flöde för alla våra kunder, oavsett företagsform. Nu kan du lägga mindre tid på deklarationsfiler och mer tid på det som verkligen betyder något för din verksamhet!
