# Exportera och arkivera data för flera år i Skatt & Bokslut

I Skatt & Bokslut kan du nu exportera och arkivera företagets data för flera år samtidigt. Genom att ladda ner en samlad fil med all dokumentation kan du enkelt uppfylla lagkrav på arkivering och säkerställa att du har tillgång till dina underlag även om du avslutar ett abonnemang.

## Förutsättningar
Denna funktion är tillgänglig för dig som använder Skatt & Bokslut (både Pro-användare och icke-interaktiva användare) samt för dig som har läsbehörighet i plattformen.

## Så här gör du för att exportera data

Följ dessa steg för att skapa en arkivfil:

1. Gå till menyn och välj **Underhåll** – **Exportera data**.
2. Under rubriken **Räkenskapsår**, klicka på rullistan för att välja vilka år du vill exportera. Du kan markera enskilda år eller välja **Alla år** för att inkludera hela företagets historik.
3. Markera kryssrutan för **Dokumentation**. Du kan föra muspekaren över informationssymbolen till höger för att se exakt vad som ingår i exporten (exempelvis skatteberäkningar, årsredovisning och avstämningar).
4. Klicka på knappen **Exportera**.
5. En förloppsindikator visas nu som visar hur långt exporten har kommit. Under tiden som filen skapas är exportknappen inaktiverad.
6. När exporten är klar visas ett meddelande och du kan klicka på **Ladda ner** för att spara ZIP-filen på din dator.

## Resultat
Du får en ZIP-fil som innehåller en strukturerad mapp för varje valt år. Inuti varje årsmapp finns undermappar för de olika perioderna. 

Varje periodmapp innehåller PDF-dokument för alla relevanta underlag. För att hålla arkivet relevant inkluderas endast perioder som innehåller automatiska kontroller, slutförda kontoavstämningar eller kommentarer.

## Felsökning

*   **Delvis genomförd export:** Om ett dokument i exporten har ett felaktigt format eller inte kan skapas, kommer du få ett meddelande om att rapporten delvis har misslyckats men ändå genererats. I ZIP-filen ersätts det felaktiga dokumentet med en vit sida som förklarar vilket dokument det gäller och vad som gick fel.
*   **Exporten misslyckas helt:** Om ett tekniskt fel uppstår under processen visas ett felmeddelande. Prova att starta om exporten. Om problemet kvarstår, kontrollera att det finns data registrerat för de valda åren.
*   **Inga PDF-filer skapas:** Exporten inkluderar endast perioder där aktivitet har registrerats (t.ex. avstämningar eller kommentarer). Om ett år saknar sådana markeringar kommer inga PDF-filer att skapas för det året.
