# Export and archive company data in {{resource:common_product_name_variable_taxation_year_end_closing}}

Here you will learn how to export and archive all documentation for your company in {{resource:common_product_name_variable_taxation_year_end_closing}}. This feature is designed to allow you to easily save your documentation in accordance with statutory archiving requirements, or to have a backup if you plan to cancel your subscription.

## Introduction
By using the export function, the platform creates a structured ZIP file containing PDF documents for the years and periods you select. The documentation is automatically organized into folders, making it easy to find the correct records afterwards.

This function is available to those with an active subscription, as well as those in read-only mode.

## Prerequisites
To be able to perform the export, you must have permission to work in the relevant client in {{resource:common_product_name_variable_taxation_year_end_closing}}.

## How to do it
Follow these steps to export your company data:

1. Navigate to the menu and select ****{{resource:Menu_title_Maintenance}}**** – ****{{resource:ExportData_title_ExportData}}****.
2. Under the heading **Export**, select the **Documentation** option.
3. In the **Financial year** dropdown, select the years you want to include in the export. If you want to export everything, select **All years**.
4. Click the **Export** button.
5. A status bar will appear showing the progress of the export. While the export is running, the **Export** button is disabled.
6. Once the export is complete, you will receive a confirmation and can download the ZIP file via a notification or directly in the view.

## Result
The export is delivered as a ZIP file. When you unzip the file, it is structured as follows:

*   **Main folders:** Named after financial years.
*   **Subfolders:** Inside each year, there are folders for each period.
*   **Documents:** In each period folder, you will find PDF files for all relevant documents (e.g., tax calculations, year-end checks, and reconciliations).

**Note:** To keep the export relevant, only periods with performed checks, account markings, or comments are included.

## Troubleshooting
If something goes wrong during the export, for example, if a specific document cannot be generated, you will receive a message about this.

*   **Export partially successful:** If most documents were created but a few failed, the ZIP file is still generated. You will then receive a notification that the export was partially successful.
*   **A specific document fails:** If a PDF cannot be created, a white page is shown in the export with information about which file it concerns and what went wrong.
*   **Export fails completely:** If a serious error occurs, an error message is displayed explaining that the file could not be created. Try restarting the export after a short while.
