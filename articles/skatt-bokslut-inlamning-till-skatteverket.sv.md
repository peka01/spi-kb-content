# Inlämning till Skatteverket

När det är dags att avsluta räkenskapsåret och skicka in företagets uppgifter gör du det smidigt under vyn för inlämning. Här får du hjälp att kontrollera att allt är rätt ifyllt innan du skickar iväg deklarationen digitalt eller skriver ut den för manuell hantering. 

Från och med beskattningsår 2025-P1 kan även handelsbolag, ekonomiska föreningar och stiftelser lämna in sina deklarationer (INK3 och INK4) helt digitalt via API, på samma enkla sätt som aktiebolag (INK2).

## Så här gör du

Innan du skickar in deklarationen gör plattformen automatiskt en **Deklarationskontroll**. Den letar efter vanliga fel eller inkonsekvenser. Om något behöver rättas visas en förklaring, och genom att klicka på **Gå till kontroll** hamnar du direkt på det fält som behöver ses över.

Du kan välja mellan tre olika sätt att lämna in deklarationen:

### Alternativ 1: Automatisk överföring via API (rekommenderas)
Detta är det enklaste sättet för dig som deklarerar **Inkomstdeklaration 2, 3 eller 4**.

1. Gå till **Årsavslut** – **Deklarationsblanketter** – **Inlämning**.
2. Välj **Inkomstdeklaration - lämna (API)** som inlämningsmetod. 
3. Kontrollera uppgifterna för uppgiftslämnaren. Om du behöver ändra något väljer du **Redigera/återställ uppgiftslämnare**.
4. Klicka på **Lämna in**. Uppgifterna förs nu över direkt till Skatteverket.
5. För att deklarationen ska vara giltig måste den signeras av behörig person. Klicka på länken till **Mina sidor** hos Skatteverket som visas på skärmen för att logga in och skriva under.
6. När det är klart kan du markera rutan **Inkomstdeklarationen är underskriven på Mina sidor** i Spiris för att hålla koll på statusen.

### Alternativ 2: Filöverföring via Skatteverkets tjänst (SRU)
Om du föredrar att använda den traditionella filöverföringen kan du ladda ner SRU-filer. Du kan enkelt växla mellan API och SRU på inlämningssidan.

1. Välj om du vill inkludera hela inkomstdeklarationen eller bara bilagorna.
2. Klicka på **Ladda ner info.sru** och **Ladda ner blanketter.sru**. Filerna sparas på din dator.
3. Gå till Skatteverkets sida för filöverföring och logga in.
4. Ladda upp de två filerna du precis sparade ner.

### Alternativ 3: Skriv ut och skicka manuellt
Om du föredrar att skicka in deklarationen per post kan du skapa en PDF.

1. Öppna **Rapportcentralen**.
2. Markera de blanketter du vill skicka under **Rapport - Deklarationsblanketter**.
3. Välj **Skapa rapport**.
4. Klicka på **Skriv ut** och välj **Ladda ner pdf**.
5. Skriv ut dokumentet, skriv under för hand och posta till Skatteverket.

## Tips
> **Digital inlämning för fler bolagsformer:** Nu kan även du som driver handelsbolag eller stiftelse använda den smidiga API-överföringen för inkomstdeklaration 3 och 4. Det sparar tid och minskar risken för manuella fel vid filhantering. Tänk på att detta gäller från beskattningsår 2025-P1.
