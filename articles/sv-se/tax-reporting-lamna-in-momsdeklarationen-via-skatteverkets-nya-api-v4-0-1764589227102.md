# Lämna in momsdeklarationen via Skatteverkets nya API v4.0

## Översikt
Den här artikeln beskriver hur du lämnar in din momsdeklaration direkt till Skatteverkets API v4.0 via Visma Bokföring & Fakturering. Denna uppdatering ger dig en snabbare och mer tillförlitlig inlämning, förbättrad felhantering samt möjlighet att göra delbetalningar direkt i systemet.

## Förutsättningar
Innan du kan lämna in momsdeklarationen, se till att du har:
*   En färdigställd momsdeklaration i Visma Bokföring & Fakturering.
*   Tillräckliga användarrättigheter för att lämna in deklarationer.
*   För att använda delbetalningsfunktionen: Uppdaterade BankID-certifikat installerade på din enhet.

## Steg
Följ dessa steg för att lämna in din momsdeklaration:

1.  **Navigera till Momsdeklarationen**
    Gå till `Bokföring & Fakturering` > `Rapporter` > `Momsdeklaration`.
    ![Screenshot: Översikt över momsdeklarationer]

2.  **Granska och förbered inlämning**
    Välj den momsperiod du vill deklarera. Granska deklarationen noggrant. Du kommer nu att se det uppdaterade `Momsdeklarationsformuläret` med nya fält.
    ![Screenshot: Momsdeklarationsformuläret]
