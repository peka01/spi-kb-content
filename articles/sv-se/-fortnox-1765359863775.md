# Integration med Fortnox i Skatt & Bokslut

## Översikt
Den här artikeln beskriver hur du integrerar Skatt & Bokslut med Fortnox för att importera bokföringsdata och skicka tillbaka bokslutsposter via API. Denna integration gör det möjligt att smidigt överföra finansiell information mellan systemen.

## Förutsättningar
*   Tillgång till Skatt & Bokslut.
*   Tillgång till Fortnox med nödvändiga behörigheter för API-integration.
*   En aktiv Fortnox-anslutning konfigurerad i Skatt & Bokslut.

## Kom igång med Fortnox-integrationen

### Lägg till Fortnox som datakälla i Startguiden

1.  Öppna Skatt & Bokslut.
2.  Navigera till **Startguiden**.
3.  Du kommer att se Fortnox som ett nytt, femte alternativ för datakälla. Placeringen kan diskuteras ytterligare och datakällorna kan grupperas om vid behov.
4.  Välj Fortnox för att påbörja konfigurationen.

### Importera bokföringsdata från Fortnox

Skatt & Bokslut stöder import av bokföringsdata från Fortnox som en SIE4-anslutning.

1.  I **Startguiden** väljer du Fortnox som datakälla.
2.  Följ instruktionerna för att etablera anslutningen.
3.  När data importeras kommer Skatt & Bokslut automatiskt att skapa bilagor för kundreskontra och leverantörsreskontra.

**Valideringar:**
Vid import från Fortnox används samma valideringar som för SIE4, inklusive:
*   Organisationsnummer
*   Filstorleksbegränsning (100 MB)
*   Hantering av saknade verifikatserier (systemet fyller i dessa)
*   Hoppa över objekt om det finns fler än 500 (för att optimera prestanda)

### Skicka tillbaka bokslutsposter till Fortnox

Du kan nu skicka bokslutsposter tillbaka till Fortnox via API:et.

1.  Slutför dina bokslutsposter i Skatt & Bokslut.
2.  Navigera till funktionen för att skicka bokslutsposter.
3.  Välj Fortnox som destination.
4.  Följ instruktionerna för att genomföra överföringen.

### Ändra importkälla

Om du behöver byta importkälla eller ändra inställningar för din Fortnox-integration:

1.  Gå till **Underhåll** > **Inställningar** > **Datakälla**.
2.  Här hittar du Fortnox listad som en av dina datakällor.
3.  Du kan nu ändra eller byta till en annan importkälla vid behov.
