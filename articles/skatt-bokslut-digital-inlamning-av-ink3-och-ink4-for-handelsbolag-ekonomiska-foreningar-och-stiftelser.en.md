# Digital submission of INK3 and INK4 for partnerships, non-profit organizations, and foundations
Starting from 2025-P1, you can submit income tax returns 3 and 4 digitally via the Swedish Tax Agency's API directly for partnerships, economic associations, and foundations. This workflow replaces the need for manual handling of SRU files and streamlines the entire tax return process. On the submission page, you can now easily choose between the new API connection or the traditional method using SRU files. The update reduces the risk of manual errors and provides you with immediate confirmation that the submission has been received.
