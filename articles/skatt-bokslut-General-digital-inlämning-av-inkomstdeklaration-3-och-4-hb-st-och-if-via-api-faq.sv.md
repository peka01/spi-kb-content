# Vanliga frågor: Digital inlämning av inkomstdeklaration 3 och 4 via API

Här hittar du svar på de vanligaste frågorna om den nya möjligheten att skicka in inkomstdeklarationer för handelsbolag, stiftelser och ideella föreningar digitalt direkt från Spiris.

### Kan jag lämna in deklarationen digitalt för handelsbolag och föreningar?
Ja, absolut! Från och med beskattningsåret 2025 (period 1) kan du skicka in inkomstdeklaration 3 (INK3) och inkomstdeklaration 4 (INK4) direkt till Skatteverket via deras API. Det innebär att du får samma smidiga flöde för dessa företagsformer som du tidigare haft för aktiebolag (INK2).

### Vilka företagsformer omfattas av nyheten?
Den digitala inlämningen via API gäller nu för:
*   Handelsbolag (HB)
*   Stiftelser (ST)
*   Ideella föreningar (IF)

### Hur väljer jag att skicka in via API?
När du är klar med din deklaration i Spiris och navigerar till sidan för inlämning, kommer du att se ett val för inlämningsmetod. Där kan du välja mellan det moderna alternativet att skicka via API eller det äldre sättet via SRU-filer. Vi rekommenderar API-lösningen eftersom den ger en snabbare och mer direkt återkoppling från Skatteverket.

### Varför ska jag välja API istället för SRU?
Att skicka via API är ett tryggare och mer effektivt sätt att deklarera. Några av fördelarna är:
*   **Direkt bekräftelse**: Du får svar direkt om inlämningen har tagits emot korrekt.
*   **Mindre manuellt arbete**: Du slipper ladda ner och ladda upp filer manuellt till Skatteverkets filöverföringstjänst.
*   **Samma flöde för alla**: Du kan använda samma enkla arbetssätt oavsett om du deklarerar för ett aktiebolag eller ett handelsbolag.

### Behöver jag ändra något i mitt abonnemang för att använda detta?
Nej, funktionen ingår som en del av de vanliga uppdateringarna i din ekonomiplattform. Så länge du har ett aktivt abonnemang som inkluderar deklarationsfunktioner kommer du att se de nya valen på inlämningssidan när det är dags för period 2025-P1.
