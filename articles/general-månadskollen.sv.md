# Få bättre insikt med den uppdaterade Månadskollen

**Månadskollen** är ett kraftfullt verktyg i Spiris som ger dig en snabb och visuell sammanfattning av hur din verksamhet presterade under den föregående månaden. Vi har uppdaterat rapporten för att ge dig ännu djupare insikter i dina digitala flöden, din likviditet och hur mycket tid du faktiskt sparar på att automatisera din bokföring.

Den största nyheten är att **Månadskollen** nu visar detaljerad statistik för både försäljningsfakturor och inköpsfakturor med fokus på digitaliseringsgrad. Du kan se exakt hur stor andel av dina dokument som hanterats helt digitalt och få en konkret uppskattning på hur mycket arbetstid du har sparat jämfört med traditionell, manuell hantering. För att du inte ska missa viktig information ligger notisen om att rapporten är klar kvar i plattformen i 14 dagar.

## Så här gör du

1.  **Hitta rapporten:** När en ny månad börjar genereras rapporten automatiskt. Klicka på symbolen för **Notiser** (klockan) i det övre högra hörnet för att se meddelandet.
2.  **Öppna sammanställningen:** Klicka på notisen **Månadsrapporten är klar**. Rapporten öppnas i ett interaktivt format direkt i webbläsaren.
3.  **Analysera din data:** Bläddra igenom rapporten för att se nyckeltal som:
    *   **Total omsättning:** Jämförelse med tidigare månader för att se trender.
    *   **Mest sålda artiklar:** Vilka produkter eller tjänster som drivit din försäljning.
    *   **Största utgiftsposter:** Vilka leverantörer eller kostnadskategorier som varit störst under månaden.
4.  **Utvärdera digitaliseringen:** Under sidorna för **Försäljning** och **Inköp** hittar du statistik för digitala fakturor. Här ser du volymen av fakturor som skickats eller tagits emot via digitala kanaler och den beräknade tidsvinsten i minuter eller timmar.

## Vad räknas som digitalt?

För att hjälpa dig att arbeta så effektivt som möjligt mäter vi de flöden som minimerar manuell datainmatning och posthantering:

*   **Försäljning:** Fakturor som skickas som e-faktura (Peppol/Autoinvoice) eller direkt till kundens digitala brevlåda i Kivra. Detta eliminerar behovet av utskrifter, kuvertering och porto.
*   **Inköp:** Fakturor som tas emot som e-faktura direkt in i systemet, via automatisk e-posttolkning (PDF-fakturor som skickas till din unika Spiris-adress) eller kvitton som fotograferats med Spiris Scanner-appen.

## Exempel på insikter du kan få

Genom att studera Månadskollen kan du identifiera konkreta förbättringsområden:

*   **Identifiera "analoga" kunder:** Om din digitaliseringsgrad på försäljning är låg kan du se vilka kunder som fortfarande får fakturor via post eller manuell e-post och bjuda in dem till e-faktura.
*   **Optimera inköpsrutiner:** Om du ser att många inköp fortfarande registreras manuellt kan det vara ett tecken på att fler i personalen behöver använda Spiris Scanner-appen för sina utlägg.
*   **Tidsbesparing:** Om rapporten visar att du sparat 5 timmar på digital hantering, är detta tid du istället har kunnat lägga på kundbearbetning eller verksamhetsutveckling.

## Tips
> **Sikta mot 100 %:** Ju högre digitaliseringsgrad du har, desto lägre blir risken för mänskliga felmatningar. I **Månadskollen** ser du potentialen – det vill säga hur mycket mer tid du skulle spara om resterande fakturaflöden också blev digitala. Använd detta som ett mål för att effektivisera din administration månad för månad.
