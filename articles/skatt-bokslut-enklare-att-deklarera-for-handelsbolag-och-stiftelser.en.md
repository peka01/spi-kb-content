## Simpler digital tax return for trading partnerships and foundations

It is now significantly easier for those filing tax returns for trading partnerships, foundations, or non-profit organizations. Previously, you had to handle SRU files manually for these types of entities, but starting from the 2025-P1 tax period, you can submit Income Tax Return 3 and 4 directly to the Swedish Tax Agency via API. This means you no longer need to download and upload files and can instead manage everything in the same smooth workflow you are used to for limited companies.

In practice, this means that on the submission page in {{resource:common_product_name_variable_taxation_year_end_closing}}, you simply choose digital submission via API instead of the traditional file selection. The data is then sent encrypted and securely, and you receive immediate confirmation that the Swedish Tax Agency has received the submission. By avoiding manual file handling, you reduce the risk of errors while saving valuable time in your tax return work.
