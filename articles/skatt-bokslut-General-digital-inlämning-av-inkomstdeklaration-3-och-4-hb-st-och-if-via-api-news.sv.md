# Nu kan du lämna in inkomstdeklaration 3 och 4 digitalt via API

Nu blir det ännu enklare för dig som arbetar med handelsbolag, kommanditbolag eller stiftelser i Spiris. Vi har nämligen öppnat upp för digital inlämning via API även för inkomstdeklaration 3 och 4, vilket innebär att du får samma smidiga flöde som du tidigare haft för aktiebolag. Istället för att hantera filer manuellt kan du nu skicka in uppgifterna direkt till Skatteverket, vilket sparar värdefull tid och minskar risken för fel under den intensiva deklarationsperioden.

När det är dags att skicka in deklarationen för beskattningsåret 2024 väljer du helt enkelt det digitala alternativet för inlämning via API på sidan för inlämning. Du har fortfarande kvar möjligheten att använda SRU-metoden om du föredrar det, men genom att använda API-inlämning får du en snabbare bekräftelse och en modernare arbetsprocess. Det här är ytterligare ett steg i att göra ditt bokslutsarbete så stressfritt och effektivt som möjligt, oavsett vilken företagsform du deklarerar för.
