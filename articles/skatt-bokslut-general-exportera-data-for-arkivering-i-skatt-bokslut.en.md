# Export data for archiving in Skatt & Bokslut

You can now easily export and archive your company's data directly from Skatt & Bokslut. This feature allows you to download all documentation as PDF files in an organized folder structure, making it easier for you to save accounting information in accordance with current legal requirements.

## What does the export contain?
When you perform an export, a zip file is created containing a folder for each selected financial year. Inside these folders are subfolders for each period, where all relevant documentation (such as reconciliations, tax calculations, and annual report documents) is saved as PDF files.

To keep the archive relevant, only periods are included where:
*   Automatic checks have been performed.
*   Account markings have been completed.
*   At least one comment has been added.

## How to do it

Follow these steps to create your archive file:

1.  Go to the menu ****{{resource:Menu_title_Maintenance}}**** and select **Export data**.
2.  Select the checkbox for **Documentation**.
3.  Select the years you want to export in the drop-down list under **Financial year**. You can choose to export a single year or select **All years** to export everything at once.
4.  Click the **Export** button.
5.  A progress bar is now displayed, keeping you updated on the progress of the export. While the export is in progress, the button is disabled.
6.  When the file is ready, you will receive a notification with a link to download the zip file.

## Result
You now have a finished zip file saved on your computer. The file is organized in a way that makes it easy to find specific documents for follow-up or auditing in the future.

## Good to know
*   **Access for everyone:** The feature is available for both active users and for those who have their subscription in read-only mode.
*   **If something goes wrong:** Should a specific PDF fail to be created due to a technical error, you will receive a notification that the export partially failed. In the zip file, the document in question is then replaced with an information page explaining what went wrong, so you can rectify it and try again.
*   **Help texts:** If you are unsure about what the different parts of the export contain, you can hover your mouse over the information icons (i-signs) next to the options for more information.
