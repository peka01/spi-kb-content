# Export all your data for smooth archiving

It is now significantly easier to secure the company's history and fulfill archiving requirements. We have added a new feature in {{resource:common_product_name_variable_taxation_year_end_closing}} that allows you to export all documentation for one or more years simultaneously. You can find the feature under the Maintenance menu and the Export data option. When you start the export, a zip file is created where all material is neatly organized into folders for each year and period, with all relevant documents saved as PDF files. This is a secure solution for those who want a local copy of their work or need to prepare material for long-term archiving.

The feature is available to all users, even if you currently only have read-only permissions in the platform. You choose which financial years to include in the export by selecting them in the list, and you can then follow the progress via a status bar while the file is being prepared. Should any document, against all odds, fail to be created, you will receive a clear notification of this, but the rest of the export will still be completed so that you get as much data as possible. Once everything is ready, you simply download the zip file and will then have a complete and searchable structure of the company's year-end closing and supporting documentation directly on your computer.
